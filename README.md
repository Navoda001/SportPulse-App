# ⚽ Football Player Explorer App  (SportPulse)
A modern **React Native + Expo** application that allows users to search football teams, view player details, and save favourites. The app uses a clean UI with gradients, quick filter chips, search functionality, and dynamic API fetching.

---

## 🚀 Features

### 🔍 Player Search  
- Search any football team (e.g., Arsenal, Barcelona).
- Results displayed with smooth UI cards.
- Quick filter chips for instant team selection.

### ⭐ Favourites System  
- Save or remove players from favourites.
- Stored in Redux Toolkit state.
- Favourites count shown in the dashboard.

### 🎨 Modern UI  
- Gradient background (Expo LinearGradient).
- Responsive layout.
- Clean cards and typography.
- Empty state and loading state screens.

### 🌐 API Integration  
- Uses TheSportsDB API to fetch team players.
- Fully asynchronous player lookup.

### 📱 Navigation  
- Built using React Navigation.
- Navigate between Home → Details → Favourites.

---

## 🛠️ Tech Stack

| Technology | Purpose |
|-----------|---------|

| **Redux Toolkit** | State management |
| **React Navigation** | Screen routing |

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
